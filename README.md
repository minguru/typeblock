# TypeBlock

AI(Claude Code)를 활용해 TypeScript + React를 공부하며 진행하는 옛날 토이 프로젝트 리메이크.

3D 블록 키보드가 키 입력과 클릭에 반응하는 타자 연습기를 구현할 예정입니다.

## 기능 (예정)

- 3D 블록 키보드 시각적 반응 (키보드 입력 / 마우스 클릭)
- 타자 연습 — 한글 / 영어 지원

## 기술 스택

- React 19 + TypeScript
- Vite
- SCSS

## 실행

```sh
npm install
npm run dev      # 개발 서버
npm run build    # 프로덕션 빌드
npm run preview  # 빌드 결과 미리보기
```
